## 🕹 操作方法

### PC
- ⬅ / ➡：移動
- ⬆ / Space：ジャンプ
- クリック：ブロックを壊す / 置く

### タブレット・スマホ
- 画面タップ：ブロックを壊す / 置く
- 画面左右タップ：移動（今後追加予定）
